# Connectly WhatsApp-like Angular Frontend

Angular 18 user messaging frontend styled like WhatsApp Web with a different professional color palette.

## User Features

- Login and register
- Chats list
- One-to-one chat
- Group chat
- Status updates
- Calls list
- Typing indicator
- Sent, delivered, and seen states
- Voice note and file attachment UI
- Archived chat
- Profile and settings

## Run

```bash
npm install
npm start
```

Open `http://localhost:4200`.
